# Exp 010: 10-Fold x 3-Seed Ensemble

## 설정

- ConvNeXt-Small, 224px (exp006과 동일 모델/설정)
- 10-Fold Stratified Split (기존 5 → 10)
- 3가지 시드 (42, 123, 777): 같은 구조, 다른 데이터 셔플
- 총 30개 모델 (10 folds x 3 seeds)
- 추론: 30 models x 5 TTA = 150개 예측 평균

## 왜?

- K-Fold가 5→10이면 각 fold의 val이 220→110으로 줄지만, 모델 수가 2배
- Seed가 다르면 같은 fold 구조여도 학습 과정이 달라져서 다양성 확보
- 30개 모델의 평균은 5개보다 극단적 예측을 훨씬 더 잘 완화

## 결과

| Seed | Fold별 Val LogLoss | Mean |
|------|-------------------|------|
| 42 | 0.049~0.060 | 0.0524 |
| 123 | 0.050~0.067 (**Fold3: 0.693 실패**) | 0.1171 |
| 777 | 0.040~0.065 | 0.0512 |

- Seed 123 Fold 3 수렴 실패 → 제외 (29개 모델 사용)
- 최고 단일 fold: Seed 777 Fold 9 (0.0404)
- 추론: 29 models x 5 TTA = 145개 예측 평균
- OOF CV: 0.0691 (단, OOF는 앙상블 효과가 축소됨)

## 관찰

- Seed 123 Fold 3에서 학습 실패 발생 (10-Fold은 val 110개라 불안정할 수 있음)
- Seed별 mean은 0.0512~0.0524로 exp006과 비슷
- 30개 모델 평균이 극단적 예측을 완화해서 실제 제출 시 더 나을 가능성 있음
